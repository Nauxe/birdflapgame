## Building and running the program with CMake
---
Clone and navigate into the repository
```bash
git clone https://github.com/Nauxe/birdflapgame
cd birdflapgame
```

Create build directory
```bash
cmake -S . -B build
```

Build and generate an executable `build/birdgame`
```bash
cmake --build build
```

Run
```bash
./build/birdgame
```
